NAME
    nopaste: command line pastebin.

SYNOPSIS
     | curl -F 'nopaste=<-' http://nopaste.us

DESCRIPTION
    add ? to resulting url for line numbers and syntax highlighting
    use this form to paste from a browser

EXAMPLES
    ~# cat bin/examplefile | curl -F 'nopaste=@-' http://nopaste.me

       http://nopaste.us/aXZI

   ~# firefox http://nopaste.me/aXZI?py#n-7
